## Local Development

1. Install docsify globally:
    ```bash
    npm i docsify-cli -g
    ```

2. Clone this repo:
    ```bash
    git clone https://github.com/Orion-ZA/Orion-Docs
    ```

3. Preview locally:
    ```bash
    docsify serve docs
    ```
The documentation will be available at `http://localhost:3000`
